# Habit Tracker

A React SPA that users can use to create, manage and track habits.

## Features

- Add habits
- Edit habits
- Delete habits
- Mark habits complete or incomplete
- Filter habits by status
- View stats
- Persist data with localStorage

## Tools Used

- React
- Vite
- React router
- CSS
- localStorage

## How to Run

```bash
npm install
npm run dev
```
